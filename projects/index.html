Projects

Control of dynamic model of two link arm using MPC

Model Predictive Control and Linear Quadratic Optimal Control of a two-link robot arm. Study of constructing dynamic models for robot manipulators and writing a controller for the two link arm model. The system's control algorithms of MPC and LQR were simulated in MATLAB and Simulink. The arm model was also visualised in Rviz using ROS and Python.





Simulation and analysis of ad-hoc drone swarm networks

The project studies communication networks of drones for deployment in disaster management and recovery. It included simulation of decentralised nodes using an ad-hoc TCP network on NS3 and NetAnim, developing an interface with ROS for simulating nodes on Gazebo, analysis of the network with nodes in T-formation, lawn mower etc. The project is yet to be published hence more information cannot be disclosed.





Development of robotic manipulator arm

Kratos - Mars Rover Project. Controlling a two link arm with inverse kinematics algorithm using Python and ROS. End effector control implemented separately. Circuit design and implementation of connecting the linear actuators, motor drivers, power inputs and micro-controller boards.





Mechanical design of Modular Robots

Mechanical design of autonomous self assembling modular robots and development of an inter-locking mechanism for the system on Solidworks. Individual units could connect and form swarms that are collectively useful for various tasks..

IoT Projects
