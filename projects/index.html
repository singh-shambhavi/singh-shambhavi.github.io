Control of dynamic model of two link arm using MPC

Model Predictive Control and Linear Quadratic Optimal Control of a two-link robot arm. The system's control algorithm on the dynamic model were simulated in MATLAB and Simulink. System also visualised in Rviz using ROS and Python.


Simulation and analysis of ad-hoc drone swarm networks

Simulation of decentralised ad-hoc TCP network of drones using Network Simulator 3. Developing an interface with ROS for simulations on Gazebo. Networks of several node formations simulated for deployment in disaster management and recovery.


Development of robotic manipulator

Kratos - Mars Rover Project. Controlling a two link arm with inverse kinematics algorithm using Python and ROS. End effector control implemented separately. Circuit design and implementation of connecting the actuators, drivers and controller board


Mechanical design of Modular Robots

Mechanical design of autonomous self assembling modular robots and development of an inter-locking mechanism for the system on Solidworks.


Review and Analysis of Heterogeneous Multi-Swarms Systems

Literature review of the communication, grouping/deployment and task allocation strategies used in multi-swarm systems Simulation of PSO algorithms for homogenous swarms and their hybrids that are suitable for heterogeneous swarms.


Simulation of Networked Euler-Lagrange Robotic Systems

Study of Networked Euler-Lagrange Robotic Systems manipulars with varying degrees of freedom. This is an ongoing project.


Design and Simulation of a Parallel Robot Manipulator

Model Predictive Control and Linear Quadratic Optimal Control of a two-link robot arm to be implemented. Control algorithm and dynamic simulated in MATLAB. Visualised in Rviz using ROS and Python.


IoT Projects

Pick and Place bot: Bluetooth controlled robot developed using Arduino. Chassis designed by laser cutting plywood and acrylic sheet. The robot had the ability to move small boxes of 6cm X 6cm dimension. Add list items. ESP32 based automatic hand dryer system. RC Plane.
